Tutorial su come simulare il sito <br>
-> Installa NODE.JS da https://nodejs.org/en/download <br>
-> Finito il download apri la cartella del progetto su VSCODE <br>
-> Seleziona "Terminal"->"New Terminal" e digita "npm init -y" <br>
    -> Se viene bloccato da powershell, apri il prompt dei comandi, naviga fino alla cartella e fai "npm init -y" <br>
    -> Tutti i comandi successivi se bloccati da VSCODE possono essere eseguiti nello stesso modo <br>
-> Scrivi nella console "npm install dotenv" <br>
-> Scrivi nella console "npm install express" <br>
-> Scrivi nella console "npm install mongodb" <br> <br>
-> Disattiva il Wi-Fi dell'università (quindi fatti hotspot), perchè blocca il collegamento col DB <br>
-> Scrivi nella console (VSCODE funziona) "node server.js" per avviare il sito <br>
-> Cerca sul browser "http://localhost:3000" <br>
-> Per terminare la connessione premi sulla console "ctrl-c" <br>
